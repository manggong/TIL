# Python

## 숫자와 문자열

### 숫자

~~~python
my_age = 25
my_next_age = my_age + 1 # 더하기
mutiply = 5 * 5 # 곱하기
divide = 30 /5 # 나누기
power = 2 ** 10 # 거듭제곱
remainder = 15 % 4 # 나머지 연산
~~~



### 문자열

- 따옴표로 감싸진 글

- 변수에 문자열을 넣는 예

  ~~~python
  my_name = 'Python'
  ~~~

- 텍스트 두개를 더하면 문자열이 이어붙여짐

  ~~~python
  text = '2015' + '1991'
  print(text)
  #결과값은 : 20151991
  ~~~

- 텍스트는 더하기만 가능하고, 다른연산은 불가능
