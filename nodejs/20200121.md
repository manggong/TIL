# Node.js



### 동기

- 요청과 결과가 동시에 일어난다. 

  => 롯데리아에서 주문을 받은 후 버거를 만든다.

  => 버거를 만드는 동안에 주문을 받지 못한다.



### 비동기

- 요청과 결과가 동시에 일어나지 않는다.

  => 롯데리아에서 캐셔와 요리사로 따로 둔다.

  => 캐셔는 주문만 받고 오더는 요리사가 처리한다.



### 블로킹

- 호출된 함수에게 제어권을 넘겨주고,  함수가 모든 작업을 마칠 때까지 대기한다.



### 논블로킹

- 호출된 함수에게 제어권을 넘겨주고,  다른 일을 처리 할 수 있다.
